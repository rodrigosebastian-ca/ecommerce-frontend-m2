# ecommerce-frontend-m2

Realizando mi primer proyecto de eccomerce
Como estoy ligado al agua purificada por una PYME de un familiar, me he puesto a trabajar en una eccomerce de agua purificada y así en algún momento poder entregar este sitio a mi familiar.
Este proyecto busca poder expandir la PYME Agua Purificada Ancestral, dando la posibilidad a los usuarios y futuros clientes de poder realizar compras como ellos estimen conveniente, ya no solo vía whatsapp sino también por redes sociales y web. Al tener un sitio web la PYME se hace mucho más vista además de dar una imagen seria y de proyección. 
Lo que contiene la web es una pincelada de lo que se ofrece y se vende, con un poco de relleno imaginativo para darle más contenido e historia al proyecto.
